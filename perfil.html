Projeto de Conclusão de Curso

TCC AQUI: Facilitando a Pesquisa Acadêmica

Para superar as dificuldades de plataformas existentes, criei o "TCC AQUI", um sistema que centraliza e organiza Trabalhos de Conclusão de Curso. Utilizando tecnologias como PHP e MySQL no back-end, a aplicação permite que usuários encontrem trabalhos de forma rápida através de filtros, favoritem os mais relevantes e recebam sugestões personalizadas, tornando a pesquisa acadêmica mais eficiente.

Tecnologias Utilizadas: PHP, MySQL, JavaScript, JQuery, HTML, CSS, Bootstrap

Link para o repositório: https://github.com/williamjmrosa/RepositorioVirtualTCC.git

Link para o site: https://tcc-aqui.com
